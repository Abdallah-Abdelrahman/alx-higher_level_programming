=======================
How to use MyList class
=======================

The class basically inherits from `list` and has on function
`print_sorted` that prints a list of `int` sorted

>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> my_list.print_sorted()
[]
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)

===========
list of int
===========

>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

==========
negatives
==========

>>> my_list.append(-1)
>>> my_list.print_sorted()
[-1, 1, 2, 3, 4, 5]

==========
float
==========

>>> my_list.append(-1.2)
>>> my_list.print_sorted()
[-1.2, -1, 1, 2, 3, 4, 5]
